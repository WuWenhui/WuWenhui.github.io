可能出现的问题

1、Initializing buffer pool, size = 128.0M InnoDB: mmap(137363456 bytes) failed; errno 12

在内存小的机器上启动时，会出现内存分配失败
解决方案：
修改 /etc/my.cnf 设置 innodb_buffer_pool_size = 8M
或者增加 swap 缓存
